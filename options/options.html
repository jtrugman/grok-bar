AI Omnibox

Address-bar shortcuts for AI chat. This extension never stores your prompts. Navigated AI URLs may still appear in browser history.

How to use

  1. In the address bar, type ai, then press Space or Tab.
  2. Type your prompt and press Enter.
  3. Optional one-shot provider prefix (omnibox only): ai claude explain MCP, ai grok summarize this, ai chatgpt write a regex.

Short prefixes like c, p, and g only apply in the address bar when followed by more words. Right-click “Ask …” always uses your default provider and never strips leading words from the selection.

Provider URLs (including Grok’s ?q=) are best-effort community integrations, not documented stable APIs. If a site changes its URL, only that provider’s config needs updating.

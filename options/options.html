Search with Grok

Normal address-bar search goes to Grok. No keyword required.

Default search (what you want)

  1. Install / reload this extension.
  2. If Chrome asks to change your default search engine, choose Grok.
  3. If it does not ask: open chrome://settings/searchEngines, find Grok, and set it as default.
  4. Type any question in the address bar and press Enter. You go to grok.com/?q=….

This uses Chrome’s search-engine override (Windows and Mac). After install, confirm Grok is selected under Settings → Search engine.

Optional: context menu & ai keyword

These do not change the default address-bar search (that is always Grok). They only affect right-click and the optional ai keyword for other providers.

Optional: multi-provider keyword

Type ai, then Space, then a prompt (or ai claude …) if you want ChatGPT / Claude / etc. without changing your default search.

Grok’s ?q= URL is a best-effort community integration, not a documented xAI API. Prompts are not stored by this extension; browser history may still record the navigated URL.
